# Calculadora Fantástica em Go (Golang) 🐹

Este é um projeto prático desenvolvido para exercitar os fundamentos da linguagem Go. Ele consiste em uma calculadora de linha de comando que demonstra conceitos essenciais de engenharia de software e a sintaxe idiomática da linguagem.

## 🚀 Funcionalidades

- **Operações Matemáticas:** Soma, Subtração, Multiplicação e Divisão.
- **Tratamento de Erros:** Verificação robusta para impedir divisões por zero, utilizando o padrão de retorno múltiplo do Go.
- **Persistência de Janela:** Função dedicada para aguardar o input do usuário antes de encerrar o processo, ideal para execução em ambientes Windows.
- **Interface Interativa:** Entrada de dados via terminal com formatação de resultados.

## 🛠️ Tecnologias e Pacotes

- **Go (Golang):** Linguagem principal.
- **Pacote `fmt`:** Para entrada e saída de dados formatada.
- **Pacote `os` e `bufio`:** Utilizados para manipulação avançada de entrada de dados (leitura de buffer).

## 📋 Pré-requisitos

Para rodar este projeto localmente, você precisa ter o Go instalado.
